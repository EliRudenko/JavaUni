# Питання 6. Функції у Python: оголошення, параметри, форми поліморфізму

## Коротка відповідь
Функції оголошуються через `def`. Параметри можуть бути позиційними, іменованими, мати значення за замовчуванням. Поліморфізм у Python динамічний: одна й та сама функція може працювати з різними типами, а перевантаження імені реалізується через «останнє визначення».

## Детально
- **Оголошення:** `def ім'я(параметри):`.
- **Параметри:**
  - позиційні — передаються за порядком,
  - іменовані — передаються як `ім'я=значення`,
  - значення за замовчуванням — `def f(x=0)`.
- **Поліморфізм:**
  - динамічний — функція може приймати різні типи,
  - «перевантаження» — у Python останнє визначення з тим же ім'ям замінює попередні (не як у C++/Java).

# Питання 17. ООП: оголошення операцій (+, -, тощо) для власних типів

## Коротка відповідь
Оператори для власних класів визначаються через спеціальні методи: `__add__` для `+`, `__sub__` для `-`, `__mul__` для `*`, `__truediv__` для `/` тощо.

## Детально
- **Навіщо:** щоб об’єкти поводилися “як числа” або інші типи.
- **Приклад:** якщо для класу визначено `__add__`, то `a + b` викликає `a.__add__(b)`.

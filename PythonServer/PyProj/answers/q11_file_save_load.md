# Питання 11. Робота з файлами: збереження даних і відновлення

## Коротка відповідь
Збереження даних виконується через відкриття файлу у режимі запису (`w` або `a`) і виклик `write()`. Відновлення — через режим читання (`r`) і метод `read()`.

## Детально
- **Запис:**
  - `open(filename, "w")` створює/перезаписує файл.
  - `write()` додає текст.
- **Читання:**
  - `open(filename, "r")` відкриває файл для читання.
  - `read()` повертає рядок з вмістом.
- **Контекстний менеджер (`with`)** гарантує закриття файлу.

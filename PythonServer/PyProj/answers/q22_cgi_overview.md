# Питання 22. CGI: загальний огляд технології, сфери використання, переваги, недоліки

## Коротка відповідь
CGI (Common Gateway Interface) — це стандарт взаємодії веб-сервера зі скриптами, які запускаються як окремі процеси. Сервер передає параметри через **environment** та стандартні потоки, а скрипт повертає HTTP-відповідь у stdout. CGI простий, але повільніший за сучасні WSGI/ASGI.

## Детально (простою мовою)
- **Як працює:** сервер отримує HTTP-запит → запускає скрипт → передає дані через змінні оточення та stdin → читає відповідь зі stdout.
- **Де використовується:** навчальні проєкти, прості скриптові сайти, інтеграція старих систем.
- **Переваги:** простота, зрозумілий механізм обміну, ізоляція запитів.
- **Недоліки:** кожен запит створює новий процес → високі накладні витрати й нижча швидкість.

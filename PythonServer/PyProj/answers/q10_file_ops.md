# Питання 10. Робота з файлами: операції з файлами та директоріями

## Коротка відповідь
Для роботи з файлами і директоріями у Python використовують `os` та вбудовану функцію `open()`. Основні операції: перевірка існування, створення, читання, запис, перейменування, отримання розміру файлу.

## Детально
- **Перевірка існування:** `os.path.exists()`.
- **Розмір файлу:** `os.path.getsize()`.
- **Створення/запис:** `open(..., mode="w")` та `write()`.
- **Читання:** `open(..., mode="r")` та `read()`.
- **Контекстний менеджер:** `with open(...) as f:` автоматично закриває файл.

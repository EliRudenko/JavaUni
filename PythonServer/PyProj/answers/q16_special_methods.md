# Питання 16. Спеціальні методи (__xxxx__) класів

## Коротка відповідь
Спеціальні (магiчні) методи `__init__`, `__str__`, `__repr__`, `__add__` тощо визначають, як об’єкти поводяться у вбудованих операціях Python (створення, друк, арифметика, порівняння).

## Детально
- **`__init__`** — конструктор.
- **`__str__`** — текстове представлення для користувача.
- **`__repr__`** — представлення для дебагу.
- **`__add__`, `__mul__`** — поведінка арифметичних операторів.

# Питання 20. Робота з БД: виконання SQL-запитів, параметри, результати

## Коротка відповідь
SQL-запити виконуються через курсор (`cursor.execute()`), параметри передаються як кортеж, результати читаються через ітерацію курсора. Для безпеки використовують параметризовані запити.

## Детально
- **Виконання:** `cursor.execute(sql, params)`.
- **Параметри:** передаються окремо (захист від SQL-інʼєкцій).
- **Отримання результатів:**
  - ітерація `for row in cursor`,
  - `fetchone()`/`fetchall()` або `next(cursor)`.
- **Prepared statements:** курсор із `prepared=True`.

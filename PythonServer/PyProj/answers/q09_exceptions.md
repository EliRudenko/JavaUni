# Питання 9. Виняткові ситуації та помилки. Оброблення винятків, блок try

## Коротка відповідь
Винятки у Python обробляються через `try/except/else/finally`. `try` містить код, який може впасти; `except` ловить помилки; `else` виконується якщо помилок не було; `finally` виконується завжди.

## Детально
- **try:** код, де може статись помилка.
- **except:** перехоплення конкретного типу помилки або узагальнено.
- **else:** запускається лише якщо помилок не було.
- **finally:** обов’язкове виконання для очищення ресурсів.

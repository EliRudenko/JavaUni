# Питання 18. ORM: відображення даних на об’єкти у Python

## Коротка відповідь
ORM (Object-Relational Mapping) — це підхід, коли записи з БД або іншого джерела даних відображаються у вигляді об’єктів Python. Це спрощує роботу з даними та дозволяє працювати у стилі ООП.

## Детально
- **Ідея:** замість роботи з «сирими» словниками чи рядками, ми створюємо об’єкти.
- **Переваги:** читабельність, інкапсуляція логіки у класах, спрощення доступу до полів.
- **Область використання:** веб-додатки, де БД містить багато таблиць, і потрібні об’єкти для моделі домену.

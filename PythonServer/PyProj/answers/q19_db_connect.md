# Питання 19. Робота з базами даних: порядок дій для підключення

## Коротка відповідь
Порядок дій: підготувати параметри доступу (host, user, password, database), викликати драйвер (наприклад `mysql.connector.connect`), отримати з’єднання, а потім курсор для запитів.

## Детально
1. **Задати параметри підключення** у вигляді словника.
2. **Викликати `connect()`** з цими параметрами.
3. **Обробити помилки** підключення.
4. **Зберегти об’єкт з’єднання** для подальших запитів.

## Де в коді
- `basics/11_db.py` — функція `connect_db()` використовує `mysql.connector.connect(**db_ini)`.
- `basics/1_2_dao.py` — клас `DataAccessor` читає параметри з JSON і підключається до MySQL.

## Куди перейти в коді
- **`basics/11_db.py`** — просте підключення.
- **`basics/1_2_dao.py`** — підключення через конфігурацію JSON.

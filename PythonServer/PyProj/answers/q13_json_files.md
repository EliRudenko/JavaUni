# Питання 13. Запис та відновлення JSON даних до файлів

## Коротка відповідь
Для запису JSON у файл використовується `json.dump()`, а для читання — `json.load()`. Це дозволяє зберігати структури Python у файлах і відновлювати їх.

## Детально
- **Запис:** `json.dump(obj, fp, ensure_ascii=False, indent=4)`
- **Читання:** `json.load(fp)`
- **Переваги:** JSON — читабельний людиною формат і зручний для інтеграції з іншими мовами.

# Питання 3. Основні структури даних Python (кортежі, словники тощо)

## Коротка відповідь
У Python основні структури даних: `list` (список), `tuple` (кортеж), `dict` (словник), `set` (множина). Вони застосовуються для зберігання колекцій, пар ключ-значення, унікальних значень тощо.

## Детально
- **list (список)** — змінювана послідовність. Потрібен, коли елементи можуть змінюватися або додаватися.
- **tuple (кортеж)** — незмінювана послідовність. Зручна, коли дані не повинні змінюватися.
- **dict (словник)** — зберігає пари `ключ -> значення`. Підходить для швидкого доступу до даних за ключем.
- **set (множина)** — унікальні значення без порядку. Використовується для усунення дублювань.

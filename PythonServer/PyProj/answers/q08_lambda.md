# Питання 8. Lambda у Python: переваги, недоліки, використання

## Коротка відповідь
`lambda` — це коротка анонімна функція. Зручно для простих операцій «на місці», але для складної логіки краще використовувати `def`.

## Детально
- **Переваги:** компактність, зручність у функціях вищого порядку (`map`, `filter`, `sorted`), швидкі прості вирази.
- **Недоліки:** обмеження до одного виразу, гірша читабельність при складній логіці.
- **Використання:**
  - як змінні-функції,
  - як інлайн-вирази,
  - як функції, що захоплюють змінні із зовнішньої області (closure).

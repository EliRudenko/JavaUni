# Питання 2. Основні типи даних Python, перетворення типів, операції з різними типами

## Коротка відповідь
У Python є базові типи: `int`, `float`, `str`, `bool`, `NoneType`. Також є контейнерні типи: `list`, `tuple`, `dict`, `set`. Перетворення типів робиться функціями `int()`, `float()`, `str()`, `bool()`. При операціях з різними типами діють правила неявного приведення (наприклад `int + float -> float`).

## Детально
- **Числа:**
  - `int` — цілі числа.
  - `float` — числа з плаваючою комою.
- **Рядки:** `str` — текстові дані.
- **Логічні:** `bool` — `True/False`.
- **Відсутність значення:** `None` (тип `NoneType`).
- **Перетворення:**
  - `int("10")` -> `10`
  - `float(42)` -> `42.0`
  - `str(True)` -> `"True"`
  - `bool(0)` -> `False`
- **Операції з різними типами:**
  - `int + float` дає `float`.
  - `str + str` — конкатенація.
  - `str * int` — повторення рядка.

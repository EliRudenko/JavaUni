# Питання 4. Оператори розгалуження та повторення у Python

## Коротка відповідь
Розгалуження в Python робиться через `if/elif/else`. Повторення — через `for` та `while`, а також через рекурсію (як альтернативу циклам).

## Детально
- **if/elif/else** — перевіряють умови та обирають гілку виконання.
- **for** — проходить по ітерабельних об’єктах (списки, рядки, генератори).
- **while** — повторює блок, поки умова істинна.
- **Рекурсія** — функція викликає сама себе (приклад у коді).

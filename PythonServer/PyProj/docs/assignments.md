# Перечень реализованных заданий

Ниже кратко указано, где реализованы задания (№1–№12).
Также для каждого задания есть отдельная папка в `docs/assignments/`,
где лежит мини-конспект по теме и список ключевых файлов.

1. **PBKDF2 (RFC2898)** — `basics/12_dao.py`, `cgi/dao/data_accessor.py`.
2. **Регистрация с датой рождения + валидация** — `basics/12_dao.py`, `cgi/dao/data_accessor.py`.
3. **Анализ маршрута `/lang/Controller/Action/Id` в HTML** — `cgi/access_manager.py`, `cgi/controllers/home_controller.py`, `cgi/views/home_index.html`.
4. **Метод `params` у HomeController + Index с ссылками** — `cgi/controllers/home_controller.py`, `cgi/views/home_index.html`.
5. **API `/order` с методами + тестовая страница** — `cgi/controllers/order_controller.py`, `cgi/views/ordertest_index.html`.
6. **Custom-Header + 403 при отсутствии** — `cgi/controllers/order_controller.py`, `cgi/static/js/site.js`.
7. **REST-ответы для `/order`** — `cgi/controllers/order_controller.py`.
8. **Тесты ошибок аутентификации** — `cgi/views/usertest_index.html`, `cgi/static/js/site.js`, `cgi/controllers/user_controller.py`.
9. **Вывод JWT header на странице тестов** — `cgi/views/usertest_index.html`, `cgi/static/js/site.js`.
10. **`cty=JWT` (nested JWT)** — `cgi/dao/helper.py`.
11. **Расширенная валидация JWT** — `cgi/dao/helper.py`.
12. **Итог тестов на `/usertest`** — `cgi/views/usertest_index.html`, `cgi/static/js/site.js`.

Скриншоты размещаются в `docs/screenshots/`.
